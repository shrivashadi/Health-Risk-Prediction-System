# 🧠 Health Risk Prediction System

A deployed Machine Learning web app that helps users evaluate their health risk based on personal health metrics.

🌐 Try it live: *https://health-risk-prediction-system.onrender.com*

---

## 🧩 What This Project Does

This application collects user health data (like age, weight, blood pressure, etc.) through a friendly web interface and predicts a health risk level using a trained Machine Learning model. The idea is to give users early insights into possible health concerns.

It combines a backend ML model with a simple frontend so that anyone can use it without coding knowledge.

---

## 🧱 How To Use Locally

1. Clone this repository  
   bash
   git clone https://github.com/your-username/Health-Risk-Prediction-System.git
   

2. Move into the project folder  
   bash
   cd Health-Risk-Prediction-System
   

3. Install required libraries  
   bash
   pip install -r requirements.txt
   

4. Launch the app  
   bash
   python app.py
   

Once running, open your browser and go to:


http://127.0.0.1:5000/
